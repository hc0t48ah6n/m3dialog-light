What if I want to use advanced features?

M3Dialog-light.js は、Material 3 デザインに基づいたモダンなモーダルダイアログを簡単に表示できる、超軽量なJavaScriptライブラリです。 依存関係は一切なく、1つのJavaScriptファイルを読み込むだけで使用できます。 PromiseベースのAPIにより、async/await を使ったシンプルで読みやすいコードを書くことができます。

M3Dialog-light.js is an ultra-lightweight JavaScript library for displaying modern Material 3 style modal dialogs. It has zero dependencies and works by simply including a single JavaScript file. The Promise-based API allows developers to write clean and readable code using async/await.

インストール

Installation

M3Dialog-light.js はビルドやパッケージ管理ツールを必要としません。 HTMLに 1行の scriptタグを追加するだけで利用できます。読み込み後、グローバルオブジェクト M3Dialog が利用可能になります。

M3Dialog-light.js does not require any build tools or dependencies. Simply include the script in your HTML file.After loading the script, the global M3Dialog object becomes available.

<script 
  src="https://hc0t48ah6n.github.io/modal/modal.js" 
  crossorigin="anonymous"></script>
                <script 
  src="https://hc0t48ah6n.github.io/m3dialog-light/m3dialog-light.js" 
  crossorigin="anonymous"></script>

基本的な使い方

Basic Usage

M3Dialog-light.js は Promise ベースの API を提供します。 async / await を使用することで、ユーザーの選択結果を簡単に取得できます。

M3Dialog-light.js provides a Promise-based API. Using async / await, you can easily handle user interactions.

modal("削除しますか？").then(r=>{
  if (r) {
    alert('YES');
  } else {
    alert('NO');
  }
})

アニメーションとUX

Animations & UX

このjavascriptはM3dialog.jsと同じ動作をします。M3Dialog.jsは、クリック地点や要素の中心から広がる拡大アニメーションを採用し、Material 3の「流動的な動き」を再現しています。

The library features a signature scale-up animation that follows Material 3 motion guidelines, providing a fluid and modern user experience.

カプセル化されたスタイル

Self-contained Styles

スタイルシートはJavaScriptの初期化時に自動的に <head> へ注入されるため、外部のCSSファイルを読み込む必要はありません。

CSS styles are injected into the DOM dynamically upon execution. No external .css link tags are required in your project.

技術仕様

Technical Specs

  • 軽量設計: 依存関係なし、純粋なJavaScriptのみ
  • Vanilla JS: Pure JS, zero dependencies.
  • 互換性: すべてのモダンブラウザに対応
  • Compatibility: Compatible with all modern evergreen browsers.
  • 言語： このjavascriptで呼び出されるモーダルの言語は日本語で固定です。その他の言語を使用したい場合はより高度な機能を利用できるM3dialg.jsをご利用ください
  • Language: The language of the modal called by
    this javascript is FIXED TO JAPANESE. If you want to use other languages, please use M3dialg.js, which has more advanced functions.
